Hot Beans Specification

This is specifications about the requirements for the certain jobs


Junior Web developer

1. Experience of core web technologies including html, css, javascript
2. Excellent verbal communication skills.

Front End developer

Demonstrate previous experience as a front-end developer, on personal or professional projects Experience with JavaScript, HTML5, CSS3 Experience with CSS pre-processors such as SASS and/or LESS is a plus Responsive web design and cross-browser compatibility Mobile first mindset Have excellent communication/collaboration and problem-solving skills Experience with version control such as GIT

Back End developer

1. Basic understanding of front-end technologies HTML, JavaScript, and CSS. Working knowledge of back-end development (configuration, administration and management of databases and servers) Server-side languages (e.g., PHP, C#, Java) Database technologies (e.g., MySQL and SQL)

Full Stack developer

1. You are expected to be a problem solver. 2. You are expected to be passionate. 3. Proficient with Visual Studio IDE. 4. Passion and aptitude for both coding and creating enterprise level solutions. 5. Be open to new ideas and flexible to change in processes and technology. 6. Proficient using source control using Git.
